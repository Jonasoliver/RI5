# Sistema de Gerenciamento de Empresa

## Contexto
Sistema desenvolvido em TypeScript que implementa um modelo de empresa com funcionários, endereços e telefones, baseado em diagrama UML. O sistema permite cadastrar uma empresa com seus dados, funcionários e contatos, e exibir essas informações de forma formatada no terminal.

## Como rodar

1. Instalar dependências:
```bash
npm install
```

2. Executar o programa:
```bash
npm start
```

O programa irá exibir os dados da empresa cadastrada, incluindo informações dos funcionários.
